Índice General - Guía de Desarrollo Java Senior

1. Conceptos Fundamentales

Principios base de programación, conceptos como ACID, SOLID, pilares del POO y fundamentos esenciales de ingeniería de software.

2. Arquitecturas de Software

Resumen de arquitecturas importantes: monolítica, microservicios, orientada a eventos, hexagonal y más.

3. Patrones de Diseño

Catálogo de patrones POO: creacionales, estructurales y de comportamiento, con explicaciones y ejemplos claros.

4. Java - Conceptos y Versiones

Funcionamiento de la JVM, manejo de memoria, multithreading, Collections Framework e innovaciones de cada versión de Java.

5. Preguntas de Entrevista Java

Preguntas y respuestas detalladas sobre el lenguaje, su ejecución, comparación de objetos y fundamentos de alto nivel.

6. Deadlocks y Concurrencia

Explicación de deadlocks, uso de synchronized y volatile, y estrategias para evitarlos en aplicaciones concurrentes.

7. Spring Framework

Conceptos clave de Spring, sus anotaciones, scopes, transacciones y el ciclo de vida de los beans.

8. Preguntas de Entrevista Spring

Cuestionario avanzado sobre Spring y Spring Boot, inyección de dependencias, anotaciones y buenas prácticas.

9. Hibernate y JPA

Explicación completa de JPA e Hibernate, sus diferencias, anotaciones principales, ciclos de vida y ejemplos de uso.

10. Preguntas sobre Bases de Datos

Preguntas avanzadas sobre bases de datos relacionales y no relacionales, optimización de consultas y replicación.
