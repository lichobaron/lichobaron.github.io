MongoDB with Java

[MongoDB](chatgpt://generic-entity?number=0) es una base de datos NoSQL orientada a documentos que almacena la información en formato JSON (internamente BSON). Es ampliamente utilizada en aplicaciones modernas por su flexibilidad y escalabilidad.

What is MongoDB?

A diferencia de las bases de datos relacionales, MongoDB no utiliza tablas ni filas. En su lugar, organiza los datos en:

  • Databases → contenedores de datos
  • Collections → conjuntos de documentos
  • Documents → objetos JSON

Why Use MongoDB?

  • Esquema flexible
  • Alta escalabilidad horizontal
  • Modelo de datos natural para APIs
  • Excelente rendimiento para lectura y escritura

Using MongoDB with Java

Java se conecta a MongoDB usando el MongoDB Java Driver, el cual permite trabajar directamente con documentos BSON.

Adding MongoDB Dependency (Maven)


<dependency>
  <groupId>org.mongodb</groupId>
  <artifactId>mongodb-driver-sync</artifactId>
  <version>4.11.0</version>
</dependency>

Basic Java Example


MongoClient client =
  MongoClients.create("mongodb://localhost:27017");

MongoDatabase db = client.getDatabase("mydb");
MongoCollection<Document> users =
  db.getCollection("users");

users.insertOne(new Document("name", "Carlos"));

MongoDB + Spring Boot

Spring Boot simplifica enormemente el uso de MongoDB gracias a Spring Data MongoDB, que elimina la mayor parte del código boilerplate.

Spring Boot Dependency


<dependency>
  <groupId>org.springframework.boot</groupId>
  <artifactId>spring-boot-starter-data-mongodb</artifactId>
</dependency>

Application Configuration


spring.data.mongodb.uri=mongodb://localhost:27017/mydb

Defining a Document


@Document(collection = "users")
public class User {

  @Id
  private String id;
  private String name;
  private String email;

}

Mongo Repository


public interface UserRepository
  extends MongoRepository<User, String> {

  Optional<User> findByEmail(String email);
}

Using the Repository


@Service
public class UserService {

  private final UserRepository repository;

  public UserService(UserRepository repository) {
    this.repository = repository;
  }

  public User save(User user) {
    return repository.save(user);
  }
}

Why Use Spring Boot with MongoDB?

  • Menos código repetitivo
  • Integración automática con Spring
  • Soporte para repositorios y queries derivados
  • Configuración mínima

Summary

MongoDB es una excelente opción para aplicaciones Java modernas. Usado junto con Spring Boot, permite construir APIs limpias, escalables y fáciles de mantener.
